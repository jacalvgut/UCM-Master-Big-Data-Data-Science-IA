# Tarea Final - Business Intelligence: Easy Loans Dashboard

## Descripción del Proyecto

Este proyecto contiene el dashboard de Tableau desarrollado para la **Tarea Final del módulo de Business Intelligence**, enfocado en el análisis de datos de **Easy Loans** (préstamos fáciles).

El dashboard permite visualizar y analizar información clave sobre préstamos, incluyendo métricas de rendimiento, análisis de clientes, y tendencias del negocio.

## Contenido del Repositorio

- **`easy_loans_jacobo_alvarez.twbx`**: Archivo de Tableau (packaged workbook) que contiene el dashboard interactivo desarrollado.

## Cómo Abrir y Visualizar el Dashboard

### Requisitos Previos

1. **Tableau Desktop** o **Tableau Reader** instalado en tu computadora
   - Tableau Desktop: Versión completa con capacidad de edición
   - Tableau Reader: Versión gratuita para visualización (solo lectura)

### Pasos para Abrir el Dashboard

1. **Instalar Tableau** (si aún no lo tienes):
   - Descarga Tableau Desktop desde [tableau.com](https://www.tableau.com/products/desktop)
   - O descarga Tableau Reader (gratuito) para solo visualizar

2. **Abrir el archivo**:
   - Haz doble clic en `easy_loans_jacobo_alvarez.twbx`
   - O abre Tableau y selecciona "Abrir" → navega hasta el archivo `.twbx`

3. **Navegar por el dashboard**:
   - El dashboard se abrirá automáticamente mostrando las visualizaciones principales
   - Utiliza los filtros y controles interactivos para explorar los datos

## Guía de Lectura del Dashboard

### Elementos Principales del Dashboard

1. **Filtros Interactivos**
   - Ubicados generalmente en la parte superior o lateral del dashboard
   - Permiten filtrar datos por diferentes dimensiones (fecha, región, tipo de préstamo, etc.)
   - Selecciona valores para actualizar todas las visualizaciones simultáneamente

2. **Visualizaciones Principales**
   - **Gráficos de barras/columnas**: Comparan métricas entre categorías
   - **Gráficos de líneas**: Muestran tendencias a lo largo del tiempo
   - **Gráficos circulares**: Representan proporciones y distribuciones
   - **Tablas**: Presentan datos detallados en formato tabular
   - **Mapas**: Visualizan datos geográficos (si aplica)

3. **KPIs y Métricas Clave**
   - Números destacados que muestran indicadores importantes
   - Pueden incluir: total de préstamos, monto total, tasa de aprobación, etc.

### Cómo Interpretar las Visualizaciones

#### Análisis de Tendencias Temporales
- Observa gráficos de líneas para identificar patrones a lo largo del tiempo
- Busca picos, valles y tendencias crecientes o decrecientes
- Identifica estacionalidad o ciclos en los datos

#### Comparaciones entre Categorías
- Utiliza gráficos de barras para comparar diferentes segmentos
- Identifica qué categorías tienen mejor o peor rendimiento
- Busca diferencias significativas entre grupos

#### Análisis de Distribución
- Los gráficos circulares muestran cómo se distribuyen los datos
- Identifica las categorías más representativas
- Observa si hay concentración en ciertos segmentos

#### Análisis Geográfico (si aplica)
- Los mapas muestran distribución geográfica de los datos
- Identifica regiones con mayor actividad o rendimiento
- Compara métricas entre diferentes ubicaciones

### Interacción con el Dashboard

1. **Hover (Pasar el mouse)**:
   - Pasa el cursor sobre cualquier elemento para ver información detallada (tooltips)
   - Los tooltips muestran valores específicos y contexto adicional

2. **Clic en elementos**:
   - Haz clic en barras, puntos o segmentos para filtrar otras visualizaciones
   - Algunos elementos pueden tener acciones adicionales configuradas

3. **Uso de filtros**:
   - Selecciona valores en los filtros para refinar la vista
   - Puedes seleccionar múltiples valores en algunos filtros
   - Usa "Limpiar filtros" para resetear la vista

4. **Navegación entre hojas**:
   - Si hay múltiples hojas/tabs, haz clic en ellos para cambiar de vista
   - Cada hoja puede mostrar diferentes perspectivas del análisis

## Estructura Típica de un Dashboard de Préstamos

El dashboard probablemente incluye análisis de:

- **Volumen de Préstamos**: Cantidad y monto total de préstamos procesados
- **Análisis de Clientes**: Segmentación por demografía, comportamiento, etc.
- **Rendimiento Financiero**: Tasas de interés, ingresos, márgenes
- **Análisis de Riesgo**: Tasas de morosidad, préstamos en riesgo
- **Análisis Temporal**: Tendencias mensuales, trimestrales o anuales
- **Análisis Geográfico**: Distribución por regiones o sucursales
- **Análisis de Productos**: Comparación entre diferentes tipos de préstamos

## Consejos para una Lectura Efectiva

1. **Comienza con el panorama general**: Observa primero las métricas principales y KPIs
2. **Usa los filtros estratégicamente**: Filtra por períodos o segmentos específicos para análisis detallados
3. **Compara diferentes vistas**: Si hay múltiples hojas, explora cada una para obtener diferentes perspectivas
4. **Identifica patrones y anomalías**: Busca tendencias consistentes o valores atípicos que requieran atención
5. **Toma notas**: Anota insights importantes mientras exploras el dashboard
