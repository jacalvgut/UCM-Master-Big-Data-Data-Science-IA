# UCM Tasks

This repository contains assignments, quizzes, and mandatory practical exercises completed during the Master's program in Big Data, Data Science & AI at the Universidad Complutense de Madrid (UCM).

## Repository Structure

The repository is organized by modules covering different topics in data science and artificial intelligence:

- **MOD01_PYTHON**: Python programming fundamentals and data manipulation
- **MOD02_SQL**: SQL database queries and relational database management
- **MOD03_NOSQL**: NoSQL databases and document-oriented data storage
- **MOD04_TABLEAU**: Data visualization and business intelligence with Tableau
- **MOD06_ESTADISTICA**: Statistical analysis and data interpretation
- **MOD07_MINERIA_DATOS**: Data mining techniques and methodologies
- **MOD08_MACHINE_LEARNING**: Machine learning algorithms and model implementation
- **MOD09_DEEP_LEARNING**: Deep learning architectures and neural networks
- **MOD10_IA_RNN**: Artificial Intelligence and Recurrent Neural Networks
- **MOD11_IA_NLP**: Natural Language Processing and text analysis

## Content

Each module contains:
- Practical exercises and assignments
- Code implementations (Python, SQL, JavaScript, etc.)
- Jupyter notebooks with analysis and solutions
- Documentation and reports (PDF format)
- Supporting materials and datasets

## Purpose

This repository serves as a portfolio of academic work demonstrating proficiency in various data science and AI techniques, from foundational programming skills to advanced machine learning and deep learning applications.
