# Evaluación de Bases de Datos NoSQL - MongoDB

## Resumen

Esta evaluación consiste en resolver **26 ejercicios** sobre MongoDB trabajando con un dataset de películas. Los ejercicios cubren operaciones CRUD básicas, actualización de documentos, filtrado y el framework de agregación de MongoDB.

La tarea incluye:
- **Ejercicio 0**: Importación del dataset
- **Ejercicios 1-23**: Ejercicios guiados sobre inserción, actualización, proyección, filtrado y agregación
- **Ejercicios 24-26**: Ejercicios libres sobre el pipeline de agregación

## Archivos de la Solución

- **`Online_Evaluacion_BBDD_NoSQL.pdf`**: Enunciado completo con la descripción de cada ejercicio
- **`script_movies.js`**: Archivo con todas las queries de solución para los 26 ejercicios
- **`movies.json`**: Dataset de películas que se importa en MongoDB
- **`jacobo_alvarez_gutierrez_bbddnosql.pdf`**: PDF de entrega con capturas de pantalla de los resultados

## Cómo Leer la Solución

1. Revisa el **`Online_Evaluacion_BBDD_NoSQL.pdf`** para entender qué se pide en cada ejercicio
2. Consulta **`script_movies.js`** donde encontrarás las queries organizadas por ejercicio (del 0 al 26)
3. Los ejercicios están numerados y algunos incluyen comentarios explicativos
4. Los ejercicios 16 y 20 crean nuevas colecciones (`actors` y `genres`) que son necesarias para ejercicios posteriores
